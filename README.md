A Python data analysis project using real-world COVID-19 data from Our World in Data.

## Features
- Single country statistics (cases, total deaths, total infection)
- Death rate comparison between multiple countries
- Correlation heatmap (infections, deaths, vaccinations and restrictions)
- Visual comparison cases per million of countries over time

## How to run
1. Install dependencies:
```
   pip install pandas matplotlib seaborn
```
2. Run the program:
```
   python covid_analysis.py
```
3. Follow the menu and enter country names in English (e.g. "Estonia", "Finland")

## Tech stack
- Python
- Pandas
- Matplotlib
- Seaborn
